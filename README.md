# Redone Events

## Features
* Event bubbling based on component hierarchy rather than DOM hierarchy.
* Drag and drop events.
* Cross-platform events.
* Better events for building desktop-grade interfaces.
* `stopPropagation` "namespaces".

## Events
* onFocus
* onBlur
* onEnter
* onLeave
* onPointerEnter
* onPointerLeave
* onPointerDown
* onPointerUp
* onPointerMove
* onScroll
* onDragStart
* onDrag
* onDragEnd
* onDropEnter
* onDrop
* onDropLeave
